## 🧠 What is VendorSnipe?

VendorSnipe is an **AI-powered procurement agent** that automates vendor discovery, comparison, and decision-making using **agentic workflows and live web interaction**.

Instead of manually researching vendors, users can simply describe their requirement — and the system:

- Understands the intent  
- Identifies relevant vendors  
- Compares pricing & strengths  
- Recommends the best fit  

💡 This is **not a chatbot** — it is an **autonomous decision-making system**.

---

## ⚡ Problem

Vendor selection today is:

- ❌ Manual and time-consuming  
- ❌ Scattered across multiple sources  
- ❌ Biased and inconsistent  
- ❌ Hard to scale for enterprises  

---

## 💡 Solution

VendorSnipe introduces an **AI Procurement Agent** that:

1. Interprets user requirements  
2. Executes multi-step reasoning  
3. Simulates vendor discovery workflows  
4. Outputs structured recommendations
